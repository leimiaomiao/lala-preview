拉了吗 v1.0

产品设计方案 · 娜美🍊

趣味化 × 社交化 × 孕期模式

🎯 产品定位

专为孕期妈妈/准爸爸设计的趣味便便记录 App。
通过游戏化和轻社交让"记💩"变得好玩不尴尬。

iOS SwiftUI Solo Builder 2-4周

📱 页面设计

1. 💩 记录中心（首页）

卡通森林背景，中央大💩按钮，上方"今天拉了吗？"

点击💩按钮 → 弹出记录面板
选 Bristol 分型 1-7（卡通💩表情）
选时长 + 心情 emoji
提交 → 💩长翅膀飞走 + 经验值浮动
按钮变绿 → "今日已拉 ✅"

趣味细节

  • Bristol 类型对应💩表情（便秘😵 正常😊 腹泻😰）
  • 连续天数显示🔥火焰
  • 每次记录随机掉落"纤维币"💎

2. 💎 成就收藏馆

展示柜风格，展示已解锁的💩徽章和称号

成就设计

  • 基础 首次记录 · 连续7天 · 连续30天
  • 特殊 深夜拉屎侠(23-6点) · 便秘克星(1-2→3-4)
  • 孕期 孕早期坚持 · 二胎妈妈 · 准爸爸关爱奖
  • 彩蛋 厕所里连开App 3次→"厕所达人"

解锁动画：宝箱开启 → 徽章旋转飞入 → 叮～🎵

3. 📊 便便日历（热力图）

GitHub 风格热力图，颜色表示便便质量：

■■ 深绿 = Bristol 3-4（完美）
■■ 浅绿 = Bristol 2,5（还行）
■■ 灰色 = 未记录
■■ 红色 = Bristol 1,6,7（异常）
  • 点击日期 → 查看详细记录
  • 长按分享当月成果（只显示颜色，隐私化）
  • 孕期模式：叠加孕周标记 + 便秘高发期提醒

4. 👥 便便伙伴（社交）

加好友

  • 扫码添加（可爱💩头像二维码）
  • 手机号邀请
  • 伴侣自动标记为"💕便便CP"

互动功能

  • 每日"戳一戳"提醒记录
  • 查看好友连续天数（不看详情）
  • 便秘时发 🆘 求助信号
  • 创建家庭便便群（最多6人）
详细记录仅自己可见 · 好友只看"拉了/没拉"+连续天数 · 可开隐身模式

5. 🤰 孕期模式

  • 按孕周推送便秘预防 tip
  • 28周+ 增加胎动记录入口
  • 产后自动切"月子模式"
  • 孕期便秘科普动画（医生审核）
  • 高纤维安全食谱推荐
  • 助消化孕妇瑜伽动作

6. ⚙️ 设置

  • 提醒时间（早中晚可选）
  • 伴侣绑定/解绑
  • 社交可见度设置
  • 孕期信息（预产期、孕周）
  • 数据导出 PDF 报告（就医用）

💾 数据模型

User id: UUID · nickname · avatar (💩 variations) pregnancyMode: Bool · dueDate: Date? partnerID: UUID? · joinedAt: Date PoopRecord id: UUID · userID · bristolType: 1-7 duration: Int(分钟) · mood: emoji timestamp: Date · notes: String? Achievement id: UUID · userID · type · unlockedAt isHidden: Bool (彩蛋) Friendship id: UUID · userID · friendID type: friend | partner | family

🔧 技术要点

  • 存储：Core Data + 敏感数据加密 + 离线优先
  • 动画：SwiftUI Animation + Lottie
  • 推送：每日提醒 · 好友戳戳 · 3天未拉警告 · 成就解锁
  • 隐私：数据本地优先 · 社交最小传输 · 支持删号
